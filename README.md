# Simple To-Do List

This is a simple to-do list application built with HTML, CSS, and JavaScript. It allows users to add tasks, filter them, and clear completed tasks.

## Features
- Add new tasks
- Filter tasks (All, Active, Completed)
- Clear completed tasks

## Getting Started
To use this application, simply open the `index.html` file in your web browser.
